# quick-open-in-new-window

VS Code エクスプローラーの右クリックメニュー拡張で、任意のパスを新しいウィンドウで開くことができます。

## 機能
- エクスプローラーで任意のフォルダーを右クリックし、「新しいウィンドウで開く」を選択すると、新しい VS Code ウィンドウでそのパスを開きます。
- 多言語対応（日本語、中国語、英語）。VS Code の言語設定に従って自動で切り替わります。
- 新しいウィンドウを開く際の通知表示を設定可能。

## インストール方法
1. [VS Code Marketplace](https://marketplace.visualstudio.com/) で `quick-open-in-new-window` を検索してインストール。
2. または `.vsix` ファイルをダウンロードし、コマンドパレットから「VSIX からインストール」でローカルインストール。

## 使い方
1. エクスプローラーで任意のフォルダーやワークスペースパスを右クリック。
2. 「新しいウィンドウで開く」を選択。
3. 設定で `quick-open-in-new-window` を検索し、通知の表示有無を変更可能。

## 多言語サポート
- 拡張機能の UI は VS Code の言語設定に応じて自動で日本語・中国語・英語に切り替わります。

## リモート開発について
- Remote-SSH や WSL などでリモート開発を行う場合、リモート側にも本拡張をインストールしてください。
