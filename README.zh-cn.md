# quick-open-in-new-window

VS Code 资源管理器右键菜单扩展，可在新窗口中打开任意路径。

## 功能特性
- 在资源管理器中右键任意文件夹，选择“在新窗口中打开”，即可在新 VS Code 窗口中打开该路径。
- 支持多语言（中文、日语、英文等），根据 VS Code 语言自动切换。
- 可配置是否显示打开通知。

## 安装方法
1. 从 [VS Code 扩展市场](https://marketplace.visualstudio.com/) 搜索 `quick-open-in-new-window` 并安装。
2. 或下载 `.vsix` 文件，在 VS Code 命令面板执行“从 VSIX 安装”进行本地安装。

## 使用方法
1. 在资源管理器中右键任意文件夹或工作区路径。
2. 选择“在新窗口中打开”菜单项。
3. 可在设置中搜索 `quick-open-in-new-window`，配置是否显示通知。

## 多语言支持
- 扩展会根据 VS Code 当前语言自动切换界面文本。
- 支持中文、日语、英文等。

## 远程开发说明
- 若使用 Remote-SSH、WSL 等远程开发，需在远程主机上也安装本扩展。
